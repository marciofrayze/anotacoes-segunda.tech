Definição

Em Haskell, Functor representa uma Typeclass capaz de realizar uma transformação. 

class Functor f where  
    fmap :: (a -> b) -> f a -> f b

Onde (a -> b) é uma função de transformação que será aplicada à cada elemento do functor f a e o resultado final será o f b.

Exemplos

Um exemplo clássico de Functor são as Listas.

Podemos fazer um map em uma lista, executando uma função de transformação para cada elemento. Como resultado, temos uma lista de mesmo tamanho, mas podendo conter elementos de algum outro tipo ou do mesmo tipo, mas com outros valores, ou até mesmo uma lista igual à inicial.
