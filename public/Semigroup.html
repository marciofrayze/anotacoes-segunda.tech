Semigoroup é uma Type class que declara a seguinte operação:

(<>)
Semigroup a => a -> a -> a

Exemplos

import Data.Semigroup

[1, 2, 3] <> [4, 5, 6] 
-- resultado: [1, 2, 3, 4, 5, 6]

"Hello " <> "World"
-- resultado: "Hello World"

Sum 2 <> Sum 3
-- resultado: Sum {getSum = 5}

Product 2 <> Product 3
-- resultado: Product {getProduct = 6}
